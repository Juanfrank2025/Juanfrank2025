### 🤖 Sistema de Automatización Inteligente con n8n
Diseño y ejecución de un flujo profesional para la gestión automática de registros y notificaciones.
- **Tecnologías:** n8n, JavaScript, Google Sheets, Slack, Gmail, Telegram.
- **Logros:**
  - Integración de formularios dinámicos con Google Sheets extrayendo y manipulando datos complejos (Arrays a Strings usando el método `.join()`).
  - Implementación de ramificaciones lógicas (IF/Switch) para envío de notificaciones multicanal.
